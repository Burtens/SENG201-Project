How to Run .jar files: 

1. Ensure that you are running Java JDK 14
	this can be downloaded from here: https://www.oracle.com/java/technologies/javase-jdk14-downloads.html
	#To check this is working, in command prompt type:
	
		java -version
	
	this should then output:
	
		java version "14.0.1" 2020-04-14
		Java(TM) SE Runtime Environment (build 14.0.1+7)
		Java HotSpot(TM) 64-Bit Server VM (build 14.0.1+7, mixed mode, sharing)
	
	if not follow this tutoral to change the java version you are running:
	
		https://superuser.com/questions/262757/changing-current-version-of-java-within-windows 
	
2. Extract Zip File


3. In command prompt change directory to extracted file
	
	This can be done by typing: 
	
		cd <file address>


4. Type
	
	For GUI Game: 

		java -jar 26358497_41654471_FarmingSimulatorGUI.jar

	For Commandline Game: 

		java -jar 26358497_41654471_FarmingSimulatorCL.jar


How to open source code in Eclipse:

1. Open eclipse and click import existing projects then click browse


2. Go to the location of the exported folder and click on GUIGame then click select folder


3. Make sure GUIGame is ticked in projects window


4. click Finish
